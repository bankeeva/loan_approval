# Loan approval predict

Данный проект посвящён задаче прогнозирования одобрения кредитной заявки на основе данных о клиенте и параметрах кредита.
Проект выполнен в рамках соревнования Kaggle Playground Series S4E10 и представляет собой задачу бинарной классификации.

Цель: построить модель машинного обучения, которая по характеристикам клиента предсказывает, будет ли кредит одобрен (1) или отклонён (0).
